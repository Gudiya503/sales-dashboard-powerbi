# 📊 Sales Dashboard (Power BI)

## 🔹 Project Overview
This project analyzes sales data using Power BI. It includes insights on sales, profit, region-wise performance, and category distribution.

## 🔹 Tools Used
- Power BI
- SQL
- Excel/CSV

## 🔹 Dashboard Features
- KPI Cards (Sales, Profit, Quantity)
- Category-wise Pie Chart
- Region-wise Bar Chart
- Monthly Sales Trend
- Top Customers Table
- Interactive Filters (Slicers)

## 🔹 Dataset
Sample sales dataset with 20 records including region, category, and profit.

## 🔹 How to Use
1. Download `.pbix` file  
2. Open in Power BI Desktop  
3. Explore dashboard with filters
